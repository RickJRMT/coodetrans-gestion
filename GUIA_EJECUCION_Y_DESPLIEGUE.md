# Guía de Ejecución y Despliegue

## Desarrollo

```bash
npm run dev
```

## Frontend

```bash
npm run dev:renderer
```

## Electron

```bash
npm run dev:electron
```

## Compilación

```bash
npm run build
```

## Portable

```bash
npm run build:portable
```

## Instalador

```bash
npm run build:installer
```

## Build firmado

```bash
npm run build:signed
```

## Build completo

```bash
npm run build:all
```
